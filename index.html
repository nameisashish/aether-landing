How It Works

Download, install, and start coding with AI in under 5 minutes.

1

Download & Install

Download the app for your OS from the download section below, or build from source with pnpm tauri dev.

Use Agent Team (advanced)

Toggle "Agent Team" for complex tasks. The Supervisor creates a plan and delegates to specialized agents — Architect designs, Coder implements, Reviewer checks, Tester validates.

Installation Guide

Aether Terminal is open-source and free. Follow these quick steps to get started on your OS.

macOS Installation

1

Download the .dmg file from the Download section above (Apple Silicon or Intel).

2

Open the .dmg and drag Aether Terminal into your Applications folder.

3

On first launch, macOS will show a security warning: "Aether Terminal Not Opened — Apple could not verify..."

Don't panic! This happens because we distribute the app for free without paying for Apple's $99/year code signing certificate. The app is 100% safe.

⚠️ Fix: "Aether Terminal Not Opened"

Method 1 (easiest): Right-click (or Control-click) the app → select "Open" → click "Open" in the dialog. You only need to do this once.

Method 2 (terminal): Open Terminal.app and run:
xattr -cr /Applications/Aether\ Terminal.app

This removes the quarantine flag. Then open the app normally.


Method 3 (System Settings): Try to open the app → Go to System Settings → Privacy & Security → Scroll down to find "Aether Terminal was blocked" → Click "Open Anyway"
Optional — Run Gemma locally via Ollama:
brew install ollama && ollama serve && ollama pull gemma2:9b
Once Ollama is running, Aether Terminal auto-detects it — no API key needed!

Windows Installation

1

Download the .exe installer from the Download section above.

2

Run the installer. Windows SmartScreen may show "Windows protected your PC".

⚠️ Fix: "Windows Protected Your PC"

Method 1 (easiest): Click "More info" → click "Run anyway".

Method 2: Right-click the downloaded .exe → Properties → Check "Unblock" at the bottom → Click Apply → Run the installer.
3

Follow the installer prompts. Aether Terminal will be added to your Start menu.

Why this happens: Code signing certificates for Windows cost $200–500+/year. Aether Terminal is open-source and free — you can review the source code yourself.

Optional — Run Gemma locally via Ollama: Download Ollama from ollama.com, then run: ollama pull gemma2:9b

Linux Installation

1

Download the .AppImage or .deb file from the Download section above.

2

For .AppImage: Make it executable and run it:

chmod +x Aether.Terminal_*.AppImage && ./Aether.Terminal_*.AppImage
3

For .deb (Ubuntu/Debian):

sudo dpkg -i Aether.Terminal_*.deb
No warnings on Linux. Linux does not have the same gatekeeper restrictions as macOS or Windows. The app runs immediately after installation.

Optional — Run Gemma locally via Ollama:
curl -fsSL https://ollama.com/install.sh | sh && ollama pull gemma2:9b
